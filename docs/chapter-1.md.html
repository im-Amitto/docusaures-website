    Build database

    SQlite3 database is the default database used when new Ruby on Rails application is created.

    $ cd tasksmanager
$ rails db:create

Created database 'db/development.sqlite3'
Created database 'db/test.sqlite3'

    Start server

    Now let's start the server.

    $ rails server

=> Booting Puma
=> Rails 5.2.2 application starting in development
=> Run `rails server -h` for more startup options
Puma starting in single mode...
* Version 3.12.0 (ruby 2.5.1-p57), codename: Llamas in Pajamas
* Min threads: 5, max threads: 5
* Environment: development
* Listening on tcp://0.0.0.0:3000
Use Ctrl-C to stop

    Open your browser and visit http://localhost:3000. You should see default welcome page.

    You can specify the port on which you want to run the server using -p option

    $ rails server -p 5000

    The above command will run server on port 5000. The default welcome page can then be accessed at http://localhost:5000.
